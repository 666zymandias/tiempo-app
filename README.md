# APP del tiempo
## Herramientas
- **Ecosistema:** React
- **Motor:** Vite
- **Lenguaje:** TypeScript
- **Estilos:** Framework TailwindCSS
- **Peticiones:** TanStack Query
- **API:** WeatherAPI

## Primeros pasos
1. Tener Node instalado, en este caso version 22.19.
2. Usar Vite para generar la estructura b√°sica de la carpeta:
   ```
   npm create vite@latest my-weather-app
   ```
3. Instalar tailwindcss:
   ```
   npm install -D tailwindcss postcss autoprefixer @tailwindcss/vite
   ```
4. Guardar API_KEY de WeatherAPI en .env
5. Inicializar proyecto en desarrollo con:
   ```
   npm run dev
   ```
